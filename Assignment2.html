3.2 PCA Q2

Why do we need dimension reduction? Select all that apply.

    1. Less data means less storage
    1. Removes noise & redundant features
    1. It always results being computationally less intensive than the original
    1. Always preserves important features of the model

Explanation for 1.1

Correct Answer is: C

Explanation:

  • A = [1 0] [2 3]
  • C = 0.2 * [1 0] [-1 5] = [0.2 0] [-0.2 1]
  • k = 0.2 * 1 = 0.2; l = 0 * 0
  • m = -0.2 + 2 = 1.8; n = 1 * 3 = 3
  • k+l+m+n = 0.2 + 1.8 + 3 = 5

Why care? It tests your ability to combine matrix inversion and multiplication, a core linear algebra skill.

Explanation for 1.2

Correct Answer is: 1,2 and 4

Explanation:

    1. By definition, a square matrix is singular when its determinant is zero. Therefore, true
    1. “Inverse not defined” means the matrix is not invertible, i.e. singular. Therefore, true
    1. The determinant of a triangular matrix is the product of the diagonal entries, not the inverse of that product. Therefore, false
    1. By definition, Iv = v. Therefore, true

Why care?
Knowing these properties is essential to recognize when matrices are invertible.
